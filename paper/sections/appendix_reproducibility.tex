\subsection{Build and Test}
\begin{Code}
cd contracts && npm install && npm test
npm run node
npm run deploy:local
cd ../web && npm install && npm run dev
\end{Code}
Run \texttt{node} and \texttt{deploy:local} in separate terminals.
Seven unit tests should pass.
Default Hardhat account \#0 receives \texttt{DEFAULT\_ADMIN\_ROLE} on deploy; scripts grant producer and auditor roles to accounts \#1 and \#2 for local demos.

\subsection{Ethereum Sepolia Deployment}
Configure \texttt{contracts/.env}:
\begin{Code}
DEPLOYER_PRIVATE_KEY=0x...
PRODUCER_ADDRESS=0x...
AUDITOR_ADDRESS=0x...
BASE_SEPOLIA_RPC_URL=
  https://sepolia.base.org
\end{Code}
Then run \texttt{npm run deploy:base-sepolia} and \texttt{npm run evaluate:gas}.
Fund the deployer via a Base Sepolia faucet before broadcasting transactions.
